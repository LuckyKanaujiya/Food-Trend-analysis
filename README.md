# 🍽️ Food Trends & Customer Preference Analysis

## 📌 Project Description
This project analyzes food trends and customer preferences in the Food & Beverage industry using Power BI dashboards.

The project focuses on analyzing customer orders, ratings, sentiment, and revenue to generate meaningful insights. It helps businesses improve decision-making, optimize product performance, and increase customer satisfaction.

## 🛠 Tools Used
- Power BI
- Excel / CSV

## 📊 Dashboards
- Food Trends Dashboard
- Sentiment Analysis Dashboard
- Customer Segmentation
- Product Insight
- Strategy Dashboard
