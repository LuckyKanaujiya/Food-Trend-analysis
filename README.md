# Food-Trends-Analysis
**#Food Trends & Customer Analysis Dashboard In FB Industries**

##  Project Description
This project focuses on analyzing food trends and customer preferences in the Food & Beverage industry using interactive Power BI dashboards.

The project transforms raw data into meaningful business insights by analyzing key factors such as customer orders, ratings, sentiment, and revenue performance. Multiple dashboards were designed to identify high-performing products, understand customer behavior, and track market trends.

The primary objective of this project is to enable data-driven decision-making by helping businesses optimize product strategies, enhance customer satisfaction, and improve overall revenue growth.

## 🛠 Tools Used
- Power BI
- Excel / CSV

##  Dashboards
- Food Trends Dashboard
- Sentiment Analysis Dashboard
- Customer Segmentation
- Product Insight
- Strategy Dashboard

  ## 📈 Key Insights
- Beverages generate the highest revenue due to frequent demand
- Products with higher ratings lead to increased sales
- Customer sentiment has a strong impact on repeat purchases
- Some cities contribute more to revenue due to regional preferences

## 🚀 Project Outcome

This project successfully transformed raw food industry data into meaningful business insights using Power BI dashboards. 

The analysis helped identify high-performing food categories, understand customer behavior, and evaluate the impact of ratings and sentiment on revenue. 

Overall, the project demonstrates how data-driven insights can support better decision-making, improve customer satisfaction, and enhance business performance in the Food & Beverage industry.
